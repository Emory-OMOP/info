# Tool for building code sets outside of Atlas

**Source**: [OHDSI Forums](https://forums.ohdsi.org/t/tool-for-building-code-sets-outside-of-atlas/25295)
**Matched keywords**: OMOP, OHDSI

This is built on top of the OMOP vocabulary limited to US vocabularies for now (to save space on my Azure DB).  I built this for people who don’t understand Atlas and also don’t know the enormous value built into the OMOP vocabulary tables. It addresses the need to quickly build code sets for feasibility studies and cohorts.  The tool requires a sign in (Microsoft or Google) but is free to use for OHDSI.  Feedback is welcome!
