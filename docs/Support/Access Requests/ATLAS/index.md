# Requesting ATLAS Access for De-Identified OMOP Data

This workflow walks you through requesting access to Emory's ATLAS instance for the **de-identified** OMOP dataset.

## Prerequisites

Before submitting your request, make sure you have the following:

- **Emory VPN** connected and working
- **Biomedical CITI training** completed within the last 3 years
- **Emory HIPAA training** completed

You can verify your VPN is working by navigating to the ATLAS instance directly:

[:octicons-arrow-right-24: Emory ATLAS Instance](https://ohdsi-atlas.emory.edu/atlas/#/){ target="_blank" }

## Submit the REDCap Access Request Form

All access requests are submitted through REDCap:

[:octicons-arrow-right-24: OIT Data Solutions Access Request Form](https://redcap.emory.edu/surveys/?s=KTPWCCDPEHFYLETP){ target="_blank" }

### Step 1: Select Resources and Systems

Fill in your name and Emory email address, then select the following options:

| Field | Selection |
|-------|-----------|
| **Which resources do you need access to?** | Databases, applications, and data lakes |
| **Which application, database, or data lake are you requesting access to?** | OMOP - Atlas |
| **Which OMOP domain do you need?** | OMOP - EHC De-Identified |

!!! tip
    Only request **de-identified** access for now. Access to identified data requires additional approvals.

![REDCap form showing the correct selections for ATLAS access to de-identified OMOP data](../../../assets/images/atlas_access_request_1.png)

### Step 2: Provide a Justification

Describe your use case for the ATLAS tool. If you are unsure what to write, you may use something like:

> *Exploring concept and cohort creation using the OMOP-CDM*

### Step 3: Upload Required Trainings

Confirm completion and upload certificates for both required trainings:

1. **Biomedical CITI training** — completed within the last 3 years. Select **Yes** and upload your certificate.
2. **Emory HIPAA training** — current. Select **Yes** and upload your certificate.

### Step 4: Review the OMOP Terms of Data Use

Read the **OMOP Terms of Data Use** agreement carefully, then provide your electronic signature.

??? example "What does the Terms of Data Use cover?"
    The agreement covers appropriate use of the OMOP database, including data security requirements, disclosure restrictions, and compliance with Emory's HIPAA security and technical controls.

### Step 5: Submit

Click **Submit** to complete your request. The OIT Data Solutions team will review and provision your access.

## Questions?
