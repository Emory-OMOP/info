# Access Requests

All access to Emory OMOP resources is managed through the OIT Data Solutions Access Request Form in REDCap. Select the workflow below that matches the resource you need:

<div class="grid cards grid-2" markdown>

-   :material-web:{ .lg .middle } __ATLAS__

    ---

    Request access to Emory's ATLAS instance for cohort building and standardized analyses against de-identified OMOP data.

    [:octicons-arrow-right-24: Request ATLAS Access](ATLAS/index.md)

-   :material-database:{ .lg .middle } __Database__

    ---

    Request direct SQL access to Emory's OMOP data lake on Amazon Redshift for custom queries using DBeaver, Python, R, or other tools.

    [:octicons-arrow-right-24: Request Database Access](Databases/index.md)

</div>
