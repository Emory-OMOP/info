# Study Agent has keeper services!

OHDSI's Study Agent continues toward agentic workflows for OMOP analytics, mirroring the direction of Emory's own OHDSI Agent. Worth tracking how Strategus integration progresses.

<!-- more -->

**Source**: [OHDSI Forums](https://forums.ohdsi.org/t/study-agent-has-keeper-services/25227)
**Matched keywords**: OHDSI

Study Agent is still under development but folks can now test services for Keeper-like concept generation, profile extraction, and row adjudication for phenotype validation. See it in action in this video.
Also, Study Agent has been tested and found to work great with locally-hosted models in Open WebUI (serving Ollama) and the new LLM Shim for a variety of cloud-hosted models and Hugging Face’s Text Embedding Inference.
