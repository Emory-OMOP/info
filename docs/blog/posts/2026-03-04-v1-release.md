# OMOP Enterprise v1.0.0 Released

Emory's OMOP Enterprise pipeline has reached its first major release. v1.0.0 includes all 13 core CDM tables with full Data Quality Dashboard coverage and automated DBT test validation.

<!-- more -->

## Highlights

- **96.6% DQD pass rate** across 2,374 automated checks
- **133 DBT tests** covering nullability, uniqueness, referential integrity, and domain validation — all passing
- **83 issues resolved** since v0.2.4, including 130 measurement plausibility fixes
- Automated Data Quality page generation pipeline — pages now update with every release

## What's Next

- Additional vocabulary coverage for admit/discharge concepts
- visit_detail foreign key resolution across clinical tables
- Continued measurement unit standardization
