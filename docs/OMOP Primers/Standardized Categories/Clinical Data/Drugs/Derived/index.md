# Derived Drug Elements

These tables aggregate individual `drug_exposure` records into clinically meaningful periods — answering "how long was the patient on this medication?" rather than "how many individual orders/administrations were recorded?"

<div class="grid cards" markdown>

-   :material-chart-timeline-variant:{ .lg .middle } **Drug Era**

    ---

    Continuous exposure periods at the ingredient level, aggregated using persistence window gap logic.

    [:octicons-arrow-right-24: Drug Era](Drug%20Era/index.md){ .md-button }

-   :material-pill-multiple:{ .lg .middle } **Dose Era**

    ---

    Periods of consistent daily dose. Useful for dose-response studies and tracking dose escalation/reduction.

    [:octicons-arrow-right-24: Dose Era](Dose%20Era/index.md){ .md-button }

</div>
