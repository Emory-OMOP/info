# OHDSI Educational Resources

Community-maintained resources for learning the OMOP Common Data Model, OHDSI tools, and observational health data science.

<div class="grid cards" markdown>

-   :material-school-outline:{ .lg .middle } **EHDEN Academy**

    ---

    Free courses on the CDM, standardized vocabularies, ETL processes, and OHDSI tools. The most comprehensive structured learning resource in the community.

    [:octicons-arrow-right-24: Browse courses](https://academy.ehden.eu/course/index.php?categoryid=all){ .md-button .md-button--primary }

-   :material-book-open-variant:{ .lg .middle } **The Book of OHDSI**

    ---

    The definitive guide to observational health data science using OHDSI methods — from CDM fundamentals to population-level estimation and patient-level prediction.

    [:octicons-arrow-right-24: Read online](https://ohdsi.github.io/TheBookOfOhdsi/){ .md-button }

-   :material-forum:{ .lg .middle } **OHDSI Forums**

    ---

    Active community discussions on CDM modeling, tool usage, vocabulary questions, and study design. The best place to get answers from the global OHDSI network.

    [:octicons-arrow-right-24: Visit forums](https://forums.ohdsi.org/){ .md-button }

-   :material-youtube:{ .lg .middle } **OHDSI Tutorials**

    ---

    Conference tutorials and workshop recordings covering data-to-evidence workflows, ETL development, vocabulary usage, and more.

    [:octicons-arrow-right-24: YouTube channel](https://www.youtube.com/@OHDSI){ .md-button }

</div>
