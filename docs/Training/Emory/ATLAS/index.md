# ATLAS Training Resources

ATLAS is the primary GUI tool in the OHDSI ecosystem. These resources will help you navigate Emory's instance, search vocabularies, build cohorts, and run analyses.

## Getting Started

Before using ATLAS, you'll need:

- [x] VPN access to Emory's network
- [x] An approved ATLAS access request ([:octicons-arrow-right-24: request access](../../../Support/Access%20Requests/ATLAS/index.md))
- [x] Completion of the [:octicons-arrow-right-24: Basic Training Pathway](../../index.md) for OMOP fundamentals

## 10-Minute Video Tutorials

The following walkthroughs are summarized from the [ATLAS 10-minute tutorial series](https://www.youtube.com/watch?v=35jtNdxwVEA). Each section covers a core ATLAS capability.

### 1. Navigating the Interface

The ATLAS home screen provides access to all major features — concept search, cohort definitions, characterizations, and more. The left sidebar is your primary navigation.

![ATLAS 10-minute tutorial overview — navigating the ATLAS interface](../../../assets/images/10min1.png)

### 2. Concept Sets and Vocabulary Search

Concept sets are reusable groups of clinical concepts (diagnoses, drugs, procedures) that form the building blocks of cohort definitions. Use the vocabulary search to find standard concepts, explore hierarchies, and build concept sets for your study.

![ATLAS 10-minute tutorial — concept sets and vocabulary search](../../../assets/images/10min2.png)

### 3. Cohort Definition Builder

Cohorts define your study population using inclusion/exclusion criteria. The cohort builder lets you combine concept sets with temporal logic, demographic filters, and event sequences — all without writing code.

![ATLAS 10-minute tutorial — cohort definition builder](../../../assets/images/10min3.png)

### 4. Characterization and Analysis Tools

Once a cohort is defined, ATLAS can characterize it — showing demographics, conditions, drug exposures, and other features of the population. These results help validate your cohort and explore patterns before running full analyses.

![ATLAS 10-minute tutorial — characterization and analysis tools](../../../assets/images/10min4.png)
