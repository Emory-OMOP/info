# General Session Tools and Tips

Practical guidance for OMOP training sessions and day-to-day collaboration at Emory.

## Zoom Features for Training Sessions

### Breakout Rooms

Breakout rooms are used during training sessions for small-group exercises — working through cohort definitions, reviewing SQL queries, or discussing study designs with peers.

### Zoom Docs

Zoom Docs can be used for collaborative note-taking during training sessions — capturing questions, sharing SQL snippets, and documenting follow-up items.
